## Summary
Added a Market Phase Control page to the admin dashboard for managing instrument trading states. The page displays all instruments with their current phase (TRADING/HALTED/PRE_OPEN/AUCTION) using colored StatusBadges, provides per-instrument Halt/Resume buttons with confirmation dialogs, and includes an emergency "Halt All Markets" button that requires typing "HALT ALL" to confirm and shows progress while halting.

## Key Decisions
1. Exported `normalizeInstruments` and `getPhaseAction` as pure functions from the page module to enable direct unit testing without React component rendering (RTL not available).
2. Added `haltInstrument` / `resumeInstrument` as new API functions with AbortSignal support, keeping the existing `haltTrading` / `resumeTrading` unchanged for backward compatibility with CircuitBreakers page.
3. Added `AUCTION` to the `InstrumentControl` status union type and `StatusBadge` color map (blue) since the task spec requires it but it was missing.
4. Halt All executes sequentially (not Promise.all) to avoid overwhelming the backend and to provide meaningful progress feedback.
