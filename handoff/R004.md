## Summary

## Key Decisions
1. **Type alias in `internal/types`.** Added `type Decimal = decimal.Decimal`
   so domain structs reference `types.Decimal` without every package importing
   the shared module. The shared module is wired with `require ...decimal v0.0.0`
   + `replace => ../shared/pkg/types/decimal`.
2. **Fields converted (per task scope):** `SecurityOrder.Price/StopPrice/AvgFillPrice`,
   `SecurityTrade.Price`, `Position.AvgCost/MarketValue/UnrealizedPnl`,
   `SettlementObligation.Price/NetAmount/AccruedInterest`, `BondDetails.ParValue`,
   `Bond.ParValue`, `CSDTransfer.SettlementAmount`.
3. **Left as float64 (intentional):** `CouponRate` (annual rate/ratio), `TickSize`,
   surveillance threshold `Value`, circuit-breaker `ReferencePrice/LastTradedPrice`
   and pct bands, index values/weights, `CustodyBalance.AvgCost` (not in the
   enumerated scope — flag for a follow-up; it is genuinely money and should
   likely be converted for consistency).
4. **Store boundary shim:** DB numeric columns are scanned into `float64` locals
   then converted with a new `decFromFloat(f)` helper (`decimal.NewFromFloat`,
   half-even round to 4dp). Insert args pass `.Float64()`. `trade_value` is now
   `trade.Price.MulInt64(int64(qty)).Float64()` (was `Price*float64(qty)`).
   `AvgFillPrice` (not persisted) set to `types.Decimal{}` (zero).
