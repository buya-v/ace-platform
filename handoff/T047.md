## Summary

Designed the complete trading web interface specification for the ACE commodity exchange. Produced a detailed spec document covering React 18 + TypeScript + Vite SPA architecture, component hierarchy, state management (Context + useReducer), real-time WebSocket integration (order book, trade feed, execution reports), order entry for 4 order types (limit, market, IOC, FOK), candlestick chart component, JWT authentication flow with in-memory token storage, responsive layout for mobile/tablet/desktop, and accessibility considerations.

## Key Decisions

1. **React Context + useReducer over Redux** — the app has two main state domains (auth and trading) which map cleanly to two contexts. Redux would add bundle size and boilerplate without proportional benefit.
2. **In-memory JWT storage** — access tokens stored in a closure variable, never localStorage/sessionStorage, to mitigate XSS token theft. Refresh tokens use httpOnly cookies.
3. **Silent refresh at 80% token lifetime** — proactive refresh prevents token expiry during active trading sessions.
4. **WebSocket auth via query parameter** — browsers cannot set custom headers on WebSocket upgrade, so JWT is passed as `?token=` param. Gateway validates during handshake.
5. **Exponential backoff with jitter** — prevents thundering herd on WebSocket reconnection. Max 10 attempts, max 30s delay.
6. **Sequence gap detection on order book** — tracks monotonic sequence numbers; gaps trigger REST snapshot fallback to ensure book consistency.
7. **TradingView lightweight-charts** — purpose-built for financial candlestick charts, <50KB gzipped, canvas-based for performance.
8. **Click-to-fill order book** — clicking a price level populates the order entry price, reducing manual input errors.
9. **Per-panel ErrorBoundary** — a crash in one panel (e.g., chart) doesn't take down the entire trading page.
10. **Decimal strings throughout** — all prices/quantities are string-encoded decimals matching the protobuf convention from T007, avoiding floating-point precision issues.
