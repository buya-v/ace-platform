## Failures

### 1. TestGatewayHealth/unknown_endpoint_returns_404

**File:** `tests/e2e/e2e_test.go:182`
**Error:** `expected status 404, got 401; body: {"error":{"code":"UNAUTHENTICATED","message":"Missing authorization token"}}`
**Root cause:** Gateway auth middleware runs before routing. Unknown paths hit auth rejection (401) before the router can return 404. This is a middleware ordering issue — auth should run after route matching, or a pre-routing 404 check should exist.
