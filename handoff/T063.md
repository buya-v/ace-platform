## Summary
Added global keyboard shortcuts to the admin dashboard. Created `useKeyboardShortcuts` hook with Ctrl+1-9 navigation, Ctrl+R refresh, Ctrl+E export, Escape close, and ? help toggle. Built a `ShortcutHelp` overlay component with dark-themed modal and CSS opacity transitions. Integrated both into `DashboardLayout` with KPI refresh wiring.

## Key Decisions
1. Used `document.addEventListener('keydown')` with cleanup on unmount for global shortcut capture.
2. Suppresses shortcuts when focus is in `<input>`, `<textarea>`, or `<select>` by checking `event.target.tagName`.
3. Extended `KPIContext` to expose a `refresh()` function so Ctrl+R can trigger KPI data refresh without a full page reload.
4. ShortcutHelp overlay uses CSS opacity transition for animate in/out, with click-outside-to-close on the overlay backdrop.
5. Tests use `renderHook` from `@testing-library/react` and simulate keyboard events via `document.dispatchEvent(new KeyboardEvent(...))`.
