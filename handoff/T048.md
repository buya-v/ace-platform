**Status:** DONE

## Summary

Built the trading web UI as a React 18 + TypeScript + Vite SPA at `src/web-ui/`. The application includes all specified features: JWT auth with in-memory token storage and silent refresh, real-time order book and trade feed via WebSocket with exponential backoff reconnection, order entry supporting limit/market/IOC/FOK order types, candlestick chart with canvas rendering and timeframe selector, position summary, margin status with utilization bar, and instrument selector with ticker data. 84 unit tests pass with 60%+ coverage on all business logic modules.

## Key Decisions

1. **Project at `src/web-ui/` not `src/trading-ui/`** — the task description specified `src/web-ui/` while T047 spec used `src/trading-ui/`. Followed the task description as authoritative.
2. **Canvas-based candlestick chart** — implemented a custom `drawCandlestick` function for full control over rendering without external charting library dependency in the build.
3. **In-memory JWT via closure** — `tokenManager` stores access token in a closure variable with scheduled refresh at 80% of token lifetime, matching the T047 spec's security requirement.
4. **WebSocketManager class** — encapsulates connection lifecycle, exponential backoff with jitter (max 30s, max 10 retries), and token injection via query parameter.
5. **CSS Modules for styling** — every component has a co-located `.module.css` file for scoped styles, no runtime CSS-in-JS.
6. **Per-panel ErrorBoundary** — each panel in the trading layout is wrapped in an ErrorBoundary so a crash in one panel doesn't take down the entire page.
7. **Polling for positions/margin/P&L** — REST polling at 5s intervals for data not available via WebSocket, with `Promise.allSettled` to tolerate partial failures.
8. **Click-to-fill order book** — clicking a price level in the order book populates the order entry price field.
9. **Decimal strings throughout** — all prices and quantities are string-encoded to match the protobuf convention and avoid floating-point precision issues.
