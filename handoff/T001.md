## Summary

Designed AWS dual-region active/passive architecture for the ACE platform.
Produced ADR-001 document (signed off) and on-premise infrastructure budget comparison.

## Key Decisions

1. **AWS ap-northeast-1 (Tokyo)** as primary — lowest latency to Mongolia
2. **ap-southeast-1 (Singapore)** as DR — warm standby, RTO 15 min, RPO 1 min
3. **EKS + Istio** for container orchestration with mTLS between all services
4. **5 EKS node groups** — system, exchange-core (CPU-optimised), app-general, data-processing (memory), spot-burst
5. **TimescaleDB self-managed on EC2** (not RDS) for tick data performance tuning control
6. **IRSA everywhere** — no long-lived credentials in pods
7. **3-tier subnet model** — public / app / data, with data tier having zero internet access
8. **Budget: $9,210/month** at launch, cap $12,000/month
