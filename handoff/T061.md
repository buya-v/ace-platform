## Summary
Added CSV export utility and print/PDF support to the admin dashboard. The CSV export function builds properly escaped CSV strings and triggers browser downloads via Blob URLs. A print stylesheet hides navigation chrome and styles tables for clean printing, and a Print/PDF button was added to the TopBar.

## Key Decisions
1. Separated `buildCSVString` as a pure, independently testable function from `exportToCSV` which handles DOM/Blob side effects — enables thorough unit testing of CSV generation logic.
2. Used `\r\n` line endings in CSV output per RFC 4180 specification.
3. Print stylesheet uses broad selectors (`[class*="sidebar"]`, `[class*="topbar"]`) to catch CSS Module-mangled class names alongside raw class names.
4. Print button uses `data-print-hide` attribute and `[class*="printBtn"]` selector to hide itself when printing.
5. Used a unicode printer icon instead of an SVG to avoid adding dependencies or a new icon component.
