# T125 — Admin UI: Surveillance, Fee Management, Reports

## Summary

Added three new operational pages to admin-ui following the established patterns (CSS Modules, pure function extraction, Vitest):

1. **Surveillance Dashboard** (`/dashboard/surveillance`) - Live alert feed table with severity color-coding (CRITICAL=red, HIGH=orange, MEDIUM=yellow, LOW=blue), filtering by severity/status, detail panel with resolve workflow. Uses `GET /api/v1/compliance/surveillance/alerts`.

2. **Fee Management** (`/dashboard/fees`) - Fee schedule viewer with summary cards (total rules, unique types, avg rate, tiers) and tabular display of fee rules with columns: fee type, tier, rate (bps), min, max, per-contract. Uses `GET /api/v1/admin/fees`.

3. **Reports** (`/dashboard/reports`) - Date picker with two report generators: Market Summary (OHLCV per instrument) and Large Trader Report (positions above threshold). Both support CSV download via existing `exportToCSV` utility. Uses `GET /api/v1/admin/reports/market-summary?date=` and `GET /api/v1/admin/reports/large-traders?date=`.
