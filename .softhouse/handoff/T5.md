## Decisions Made

- **56px collapsed sidebar**: Standard icon-only sidebar width that accommodates 16px icons with comfortable 20px horizontal padding.
- **`display: none` on text spans**: Preferred over `font-size: 0` trick for better accessibility and screen reader compatibility.
- **No horizontal scroll on body**: All layout changes ensure `margin-left: 56px` + content fills remaining width without overflow.
- **Clock and print button hidden on mobile**: These are low-priority elements; hiding them prevents the topbar from becoming cramped on 375px viewports.
- **Activity time hidden on tablet**: The `min-width: 80px` column takes significant space; removing it allows actor/badge/description to fit.
