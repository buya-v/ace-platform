## Decisions Made

- `useTenant()` throws if called outside `TenantProvider`; all three pages are already inside the provider in the app tree, so no try/catch needed.
- DataGrid is hidden (not shown with empty data) when `currentTenant` is null — prevents confusing "no data" message before a tenant is chosen.
- The "Select a tenant" message uses a plain `<p>` tag with no special styling to keep changes minimal and non-breaking.
- Template literal `\`Securities Instruments${currentTenant ? \` — ${currentTenant.name}\` : ''}\`` avoids conditional JSX for the title and is type-safe.
