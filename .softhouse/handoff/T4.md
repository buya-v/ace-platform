## Decisions Made

- Used `var(--text-muted)` for the icon color and `var(--text-secondary)` for the text, matching the project's CSS custom property system rather than hardcoded hex values (e.g., `#999`, `#666`). This ensures the empty state respects theme changes automatically.
- 32px icon size was specified in the task; it is passed via the `size` prop directly in JSX rather than CSS to keep it explicit and overridable by future callers.
- The `aria-hidden="true"` attribute is on the `<svg>` element (not a wrapper), matching how other decorative icons in the codebase are typically handled.
- The `<td className={styles.empty}>` wrapper is retained for valid table HTML; the flex layout lives in the inner `<div>` to avoid CSS grid/flexbox conflicts with `<td>` in some browsers.
