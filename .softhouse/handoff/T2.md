## Decisions Made

1. Handler test servers follow the factory pattern in `handlers_surveillance_test.go` / `handlers_audit_test.go` — fresh in-memory stores, matching engine, `TenantMiddleware([]string{testTenant})`, `httptest.NewServer`.

3. `TestShortSellWithLocate` creates the locate via HTTP then approves it directly via store to keep the test focused on the order submission path (not the approve HTTP path, which has its own test).
4. `TestGiveUpStore_Create_Accept_Reject` exercises `List(firmID)` inline rather than adding a separate `TestGiveUpStore_ListByFirm`.
5. `TestRejectGiveUp_WithoutReason` confirms the handler accepts a nil body (reason is optional per the implementation).
