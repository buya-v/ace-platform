## Decisions Made
- Each test has its own httptest.NewServer (self-contained, no shared state)
- /api/v1/auth/me mock included in all create/update tests that use withAttribution (T503 feature)
- Pre-existing `TestExecutor_Help` was already updated by linter to use "Trading & Instruments" (not "Trading Controls"); it passes as-is
- Used body capture pattern (json.NewDecoder) for POST/PUT tests to verify exact payload fields sent to gateway
- Error path tests (API Error suffix) verify ❌ reply without requiring body capture
