## Key Decisions

- Corporate action types use CA_ prefix (CA_DIVIDEND, CA_STOCK_SPLIT) matching Go consts
- FRC report query parameter is `type` (not `report_type`) per handler source
- Settlement query requires at least one of `date` or `status` per handler validation
